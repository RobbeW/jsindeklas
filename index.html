    Formularium

    • console.log("tekst") - print tekst naar de console
    • if (voorwaarde) { ... } - ALS-functie
    • else if (voorwaarde) { ... } - ANDERS ALS-functie
    • else { ... } - ANDERS-functie
    • for (var teller = start; teller kleiner dan eind; teller++){...} - begrensde herhaling
    • while (voorwaarde) { ... } - voorwaardelijke herhaling
    • moveForward(afstand) – vooruit bewegen
    • moveBackward(afstand) – achteruit bewegen
    • turnRight(hoek) – draai naar rechts (graden)
    • turnLeft(hoek) – draai naar links (graden)
    • penUp() – pen omhoog
    • penDown() – pen omlaag
    • setPenColour(r, g, b) – verander kleur
    • setPenColour("naam") – verander kleur met naam
    • randomColour() – geeft een willekeurige penkleur terug (als string)
    • goTo(x, y) – ga naar positie (x, y)
    • showGrid(stapgrootte) – raster tonen
    • setLineWidth(breedte) – lijndikte
    • resetTurtle() – reset alles
    • clearCanvas() – wis tekening
    • writeText("tekst") – tekst schrijven

    Kleuren (naam & RGB)

    Gebruik:setPenColour(r, g, b) of setPenColour("naam")
    Voorbeeld: setPenColour("crimson"); of setPenColour(220,20,60);
